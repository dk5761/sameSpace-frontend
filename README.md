# Samespace Assignment

## Installation

Install the app with pnpm

```bash
  ct into project directory
  pnpm install
  pnpm run dev
```

## Tech Stack

- React.JS
- Tailwind CSS + ShadCN/UI
- GraphQL + ApolloClient
- Framer Motion
- pnpm + vite.
